# spearwolf-js

A typescript library for my 2.5d browser game dev experiments based on [three.js](https://threejs.org/)

## Development Setup

You will need [Node.js](https://nodejs.org/) **version v10+**.

After cloning the repo, run:

```sh
$ npm install
```

To run all tests:

```sh
$ npm test
```

To start the examples, run:

```sh
$ npm start
```
